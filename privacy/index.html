Política de Privacidade

Esta Política de Privacidade descreve como o aplicativo Paghelp.me trata as informações dos usuários durante o uso de suas funcionalidades de gerenciamento de chaves PIX, criação de QR codes e links de pagamento.

Data de Vigência: Esta política é válida a partir de 28 de setembro de 2024.

1. Funcionamento Offline e Coleta de Dados
O Paghelp.me foi desenvolvido com foco na privacidade total dos usuários. O aplicativo funciona 100% offline, o que significa que:
• Nenhuma informação pessoal é enviada para servidores externos
• Todas as suas chaves PIX são armazenadas exclusivamente no seu dispositivo
• Não realizamos coleta, processamento ou compartilhamento de dados pessoais
• Não há cadastro, login ou qualquer forma de identificação do usuário
• O aplicativo não possui conexão com a internet para funcionar

2. Armazenamento Local de Dados
Todas as informações que você insere no Paghelp.me (chaves PIX, nomes, descrições) são armazenadas exclusivamente no armazenamento local do seu dispositivo:
• Os dados ficam apenas no seu celular ou tablet
• Não temos acesso a essas informações
• Os dados não são sincronizados com nuvem
• Você tem controle total sobre suas informações
• A exclusão do app remove permanentemente todos os dados

3. Compartilhamento e Transmissão de Dados
O Paghelp.me não compartilha, transmite ou processa dados de usuários:
• Não há comunicação com servidores externos
• Não utilizamos serviços de análise ou publicidade
• Não compartilhamos dados com terceiros
• Não há rastreamento de uso ou comportamento
• As únicas informações que saem do app são aquelas que você escolhe compartilhar manualmente (QR codes, links de pagamento)

4. Segurança e Proteção dos Dados
A segurança dos seus dados é garantida pelo design offline do aplicativo:
• Não há transmissão de dados pela internet
• Os dados ficam protegidos pelos mecanismos de segurança do seu dispositivo
• Opção de autenticação biométrica (impressão digital/Face ID) para acesso ao app
• Não armazenamos dados em servidores que possam ser comprometidos
• A criptografia dos dados fica por conta do sistema operacional do seu dispositivo

5. Seus Direitos e Controle dos Dados
Como usuário do Paghelp.me, você tem controle total sobre suas informações:
• Acesso: Você pode visualizar todos os dados armazenados no app
• Edição: Pode modificar ou excluir qualquer informação a qualquer momento
• Exclusão: Pode remover dados individuais ou desinstalar o app para exclusão completa
• Portabilidade: Pode exportar suas chaves PIX para outros aplicativos
• Não dependemos de consentimento para processamento, pois não processamos dados externamente

6. Permissões do Aplicativo
O Paghelp.me solicita apenas as permissões essenciais para seu funcionamento:
• Armazenamento: Para salvar suas chaves PIX no dispositivo
• Biometria: Para proteger o acesso ao app (opcional)
• Câmera: Para ler QR codes PIX (opcional)
• Não solicitamos acesso à internet, localização, contatos ou outras informações sensíveis

7. Alterações na Política de Privacidade
Esta Política de Privacidade pode ser atualizada ocasionalmente para refletir melhorias no aplicativo ou mudanças legais:
• Alterações significativas serão comunicadas nas notas de versão do app
• A data de vigência sempre será atualizada
• O uso continuado do aplicativo após alterações constitui aceitação dos novos termos
• Você sempre pode consultar a versão mais atual em: paghelpme.github.io/privacy
