Política de Privacidade

Data de Vigência: Esta política é válida a partir de 28 de setembro de 2024.

Esta Política de Privacidade descreve como o aplicativo Paghelp.me trata as informações dos usuários durante o uso de suas funcionalidades de gerenciamento de chaves PIX, criação de QR codes e links de pagamento.

1. Funcionamento Offline e Coleta de Dados

O Paghelp.me foi desenvolvido com foco na privacidade total dos usuários. O aplicativo funciona 100% offline, o que significa que:

  • Nenhuma informação pessoal é enviada para servidores externos
  • Todos os dados ficam armazenados localmente no seu dispositivo
  • Não coletamos, processamos ou compartilhamos dados pessoais
  • Não temos acesso às suas chaves PIX, QR codes ou informações financeiras

2. Armazenamento de Dados

Todas as informações inseridas no aplicativo são armazenadas exclusivamente no dispositivo do usuário:

  • Chaves PIX: Ficam no armazenamento local do dispositivo
  • Configurações: Preferências salvas localmente
  • Dados biométricos: Processados apenas pelo sistema operacional do dispositivo
  • Histórico de uso: Mantido apenas localmente, se habilitado

3. Uso de Biometria

O recurso de autenticação biométrica (impressão digital/Face ID) quando habilitado:

  • É processado exclusivamente pelos sistemas de segurança do dispositivo
  • O Paghelp.me não tem acesso aos dados biométricos
  • Utilizamos apenas APIs do sistema operacional para validação
  • Os dados biométricos permanecem no enclave seguro do dispositivo

4. Segurança

Máxima Segurança Garantida:

  • Funcionamento 100% offline elimina riscos de vazamento de dados
  • Nenhuma conexão com servidores externos
  • Dados criptografados no armazenamento local
  • Autenticação biométrica opcional para acesso ao app
  • Controle total do usuário sobre seus dados

5. Não Compartilhamento de Dados

O Paghelp.me jamais compartilha dados dos usuários porque:

  • Não coletamos dados pessoais
  • Não temos servidores para armazenar informações
  • Funcionamos completamente offline
  • Respeitamos totalmente sua privacidade

6. Conformidade com a LGPD

O Paghelp.me está em total conformidade com a Lei Geral de Proteção de Dados (LGPD) porque:

  • Não processamos dados pessoais
  • O usuário tem controle total sobre suas informações
  • Dados permanecem sempre com o titular
  • Não há transferência ou compartilhamento de dados

7. Exclusão de Dados

Para excluir completamente seus dados:

  • Desinstale o aplicativo do seu dispositivo
  • Ou use a função "Limpar Dados" nas configurações do app
  • Todos os dados serão removidos permanentemente do dispositivo

Compromisso com sua Privacidade:
O Paghelp.me foi criado com o princípio de que seus dados financeiros devem permanecer sempre sob seu controle total. Por isso, escolhemos o funcionamento offline como nossa arquitetura principal, garantindo que você tenha máxima privacidade e segurança.
